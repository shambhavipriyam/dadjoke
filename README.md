# dadjoke
Read dad jokes while you code in R

## Installation
You can install `dadjoke` from github with:
````
# install.packages("devtools")
devtools::install_github("shambhavipriyam/dadjoke", force = TRUE)
````

To load the package:
````
library(dadjoke)
````

## Examples
````
library(dadjoke)

dadjoke()
#> 
#> What's brown and sticky?
#> 
#> A stick.
#> 
````
